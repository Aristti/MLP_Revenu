About the Project

Income prediction project developed as part of Machine Learning 2

Project Objective

This project involves coding, applying, and designing a multilayer perceptron from scratch. The resulting application is a web app capable of predicting individual income based on demographic and professional features. It serves as a decision-support tool for public policy makers, financial institutions, and HR services.

Main Features

  • Loading user data through a form
  • Income prediction using a custom-built MLP

Technologies & Resources

  • Python (Pandas, Numpy)
  • Flask for backend web framework
  • HTML/CSS and JavaScript for the interface
